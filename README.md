# Android-Store-App-Java
Java (MVVM) Android mağaza uygulaması.

---
## Özellikler
* Üyelik oluşturma
* Alışveriş sepetine ürün ekleme
* Sipariş verme
* Geçmiş siparişleri görüntüleme

---

## Kullanılan Teknolojiler
* Retrofit
* Dagger2
* RxJava
* Glide
* Firebase Authentication
* Firestore Database
* ViewModel
* DataBinding
* SharedPreferences
